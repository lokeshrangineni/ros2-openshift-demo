Open-RMF Hotel Demo

Multi-fleet robot orchestration on OpenShift — 3 fleets, 4 robots, lifts & doors

Demo Details

Environment3-level hotel (lobby + 2 guest floors)
FleetstinyRobot (patrol) · deliveryRobot (delivery) · cleanerBotA (cleaning)
Robots4 total — tinyBot_1, deliveryBot_1, cleanerBotA_1, cleanerBotA_2
Infrastructure2 lifts · multiple doors · charging stations
NavigationSlot car plugin (lightweight, no GPU)
PlatformOpenShift · ROS 2 Jazzy · Gazebo (headless)

Access Points

  • Dashboard — openrmf-dashboard route (task dispatch, fleet status, map)
  • RViz2 / noVNC — openrmf-vnc route via HTTP (real-time robot visualization)
  • API — openrmf-api route (REST: /fleets, /tasks, /building_map)
