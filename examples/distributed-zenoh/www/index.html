ROS2 + Gazebo Distributed Demo on OpenShift

This is the Gazebo simulation pod of a distributed ROS2 deployment. Nav2 (navigation) runs in a separate pod, connected via Zenoh.

View the Gazebo Simulation

Click the button below to open the Gazebo GUI in your browser via noVNC. You will see the full 3D simulation of a TurtleBot3 in a sandbox world.

Open Gazebo Visualization (noVNC)

What is Running

  • This pod (gazebo-sim): Gazebo Harmonic, ros_gz_bridge, robot_state_publisher, VNC visualization
  • Separate pod (robot-nav): Nav2 full navigation stack (AMCL, planner, controller)
  • Communication: zenoh-bridge-ros2dds sidecars bridge ROS2 topics between pods over TCP
  • TurtleBot3 Waffle with LiDAR, IMU, and camera sensors

Architecture

Pod: gazebo-sim                    Pod: robot-nav
  ├── Gazebo server (physics)        ├── AMCL (localization)
  ├── ros_gz_bridge (Gz↔ROS2)       ├── planner_server
  ├── robot_state_publisher          ├── controller_server
  ├── Xvfb + x11vnc + noVNC         ├── bt_navigator
  └── zenoh-bridge (router)          └── zenoh-bridge (peer)
