ROS2 + Gazebo Demo on OpenShift

This pod runs TurtleBot3 navigation simulation (Nav2 + Gazebo Harmonic) on OpenShift with GPU-accelerated rendering.

View the Gazebo Simulation

Click the button below to open the Gazebo GUI in your browser via noVNC. You will see the full 3D simulation of a TurtleBot3 in a sandbox world.

Open Gazebo Visualization (noVNC)

What is Running

  • ROS2 Jazzy + Nav2 (full navigation stack)
  • Gazebo Harmonic (GPU-accelerated via NVIDIA A10G)
  • TurtleBot3 Waffle with LiDAR, IMU, and camera sensors
  • noVNC for browser-based visualization on port 6080

Architecture

OpenShift Pod
  ├── Xvfb (virtual display :99)
  ├── Gazebo GUI (renders to Xvfb)
  ├── x11vnc (captures display → VNC)
  ├── noVNC/websockify (VNC → browser)
  ├── Nav2 (autonomous navigation)
  └── ROS-Gazebo bridge (sensor data)
